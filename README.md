# How to develop

```
git clone git@github.com:ilime/Petal.git && cd Petal/
yarn && yarn run bundle:dev+main && yarn run dev:hot
// Open a new terminal
yarn start
```
# How to build

```
chmod +x build.sh
./build.sh
```
