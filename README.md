# Use it

# How to develop

```
git clone git@github.com:SandStorms/Petal.git && cd Petal/
npm install && npm run bundle:src+main
npm start
```
# How to build

```
chmod +x build.sh
./build.sh
```

# How to contribute

Pull a request and open a issue to describe your changes
