## Use it

Go to [release page](https://github.com/ilime/Petal/releases)

## How to develop

```sh
git clone git@github.com:ilime/Petal.git && cd Petal/
yarn && yarn start

// Open a new terminal
yarn build:electron:dev
yarn start:electron
```

## How to build

```sh
chmod +x build.sh
./build.sh
```
