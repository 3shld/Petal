# Use it

Go to [release page](https://github.com/ilime/Petal/releases)

# How to develop

```
git clone git@github.com:ilime/Petal.git && cd Petal/
npm install && npm run bundle:dev+main && npm run dev:hot
// Open a new terminal
npm start
```
# How to build

```
chmod +x build.sh
./build.sh
```
